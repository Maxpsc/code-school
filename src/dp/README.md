# 动态规划系统学习教程

## 📚 教程概述

本教程通过 **滑雪问题** 和 **背包问题** 两个经典案例，帮助你系统掌握动态规划。

## 🎯 学习目标

1. 理解动态规划的核心思想
2. 掌握状态定义和状态转移方程
3. 学会记忆化搜索和动态规划两种方法
4. 掌握空间优化技巧

## 📖 学习路径

### 第一步：理论基础
阅读 `01-理论基础.md`，理解动态规划的核心概念

### 第二步：滑雪问题
- 阅读 `02-滑雪问题.md`
- 学习 `skiing.ts` 代码实现

### 第三步：背包问题
- 阅读 `03-背包问题.md`
- 学习 `knapsack.ts` 代码实现

### 第四步：总结提升
阅读 `04-总结与练习.md`，巩固所学知识

## 💡 使用建议

1. **按顺序学习**：从理论基础开始，逐步深入
2. **边看边想**：看代码时思考为什么这样写
3. **动手实践**：尝试修改代码，观察结果变化
4. **总结归纳**：学完后总结动态规划的解题模板
